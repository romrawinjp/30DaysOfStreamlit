# Day 1 of Streamlit

1. Setting up `conda` environment
        
        conda create -n stenv python=3.9

2. Activate that conda environment

        conda activate stenv

3. Install `Streamlit` library

        pip install streamlit

4. Launch the `Streamlit` demo app

        streamlit hello
